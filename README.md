# web_befree

## Описание
Веб-сайт магазина одежды befree.

## Используемые технологии
- **Backend**: PHP
- **Frontend**: HTML, CSS
- **База данных**: MySQL (через phpMyAdmin)
- **Серверная среда**: XAMPP/LAMPP
- **Сессии**: PHP Sessions
- **Cookies**: Используются для входа

## Установка
1. Установите XAMPP/LAMPP
2. Склонируйте репозиторий в `htdocs/`
3. Импортируйте БД (php-web.sql) через phpMyAdmin
